(E) Manage Mobs
    (E) Create Mob
        (S) Game Master --> Select Multiple Tokens
        and (S) Game Master --> Group Tokens Into Mob
        and (S) Game Master --> Display Mob Creation Confirmation
    (E) Edit Mob
        (S) Game Master --> Add Tokens To Mob
        and (S) Game Master --> Remove Tokens From Mob
    (E) Select Strategy
        (S) Game Master --> Display Available Strategies
        and (S) Game Master --> Assign Strategy To Mob
    or
        (S) Game Master --> Select Attack Most Powerful Target Strategy
        or (S) Game Master --> Select Attack Weakest Target Strategy
        or (S) Game Master --> Select Defend Leader Strategy
        or (S) Game Master --> Select Attack Most Damaged Person Strategy
    (E) Execute Mob Actions
        (S) Game Master --> Click Mob Token To Command
        and (S) Game Master --> Determine Target From Strategy
        and (S) Game Master --> Execute Attack Action
    or
        (S) Game Master --> Move To Target For Melee Attack
        or (S) Game Master --> Execute Area Attack
    (E) Spawn Mob From Template
        (S) Game Master --> Select Mob Template
        and (S) Game Master --> Spawn Mob From Actors
        and (S) Game Master --> Configure Spawned Mob
