(E) Manage Mobs
    (E) Create Mobs
        (S) Game Master --> Create mob from selected minion tokens
        and (S) Game Master --> Group minions into mob
    or
        (S) Game Master --> Spawn mob from multiple actors
        or (S) Game Master --> Spawn mob from mob template
    (E) Edit Mobs
        (S) Game Master --> Add minion to existing mob
        and (S) Game Master --> Remove minion from mob

(E) Apply Strategies to Mobs
    (E) Select Strategy
        (S) Game Master --> Apply strategy to mob
    (E) Target Selection Strategies
        (S) Game Master --> Determine target based on aggression rules
        or (S) Game Master --> Determine target based on power effects
        or (S) Game Master --> Attack most powerful target
        or (S) Game Master --> Attack weakest target
        or (S) Game Master --> Defend leader
        or (S) Game Master --> Attack most damaged target

(E) Execute Mob Actions
    (E) Coordinate Attacks
        (S) Game Master --> Execute coordinated attack for all minions in mob
        and (S) Game Master --> Perform area attack with mob
    (E) Handle Movement and Range
        (S) Game Master --> Move minions to target area based on range
        and (S) Game Master --> Attack fleeing target
