(E) Manage Mobs
    (E) Create Mob
        (S) Game Master --> Group minions into mob
        and (S) Game Master --> Select minion tokens to include in mob
    (E) Spawn Mob from Actors
        (S) Game Master --> Spawn mob from actor templates
        and (S) Game Master --> Create mob templates

(E) Select Strategy
    (E) Choose Strategy
        (S) Game Master --> Select attack most powerful target strategy
        or (S) Game Master --> Select attack weakest target strategy
        or (S) Game Master --> Select defend leader strategy
        or (S) Game Master --> Select attack most damaged target strategy

(E) Execute Combat
    (E) Choose Target
        (S) Game Master --> Choose target based on strategy
        or (S) Game Master --> Apply different ways of choosing target
    (E) Execute Attack
        (S) Game Master --> Execute melee attack
        or (S) Game Master --> Execute ranged attack
        or (S) Game Master --> Move to target area for attack
        or (S) Game Master --> Execute area attack
        or (S) Game Master --> Attack fleeing target
